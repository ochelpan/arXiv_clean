Entanglement-assisted continuous-variable concatenated codes for encoding qubits or oscillators

Authors: Nihar Ranjan Dash, Sanjoy Dutta, R. Srikanth, Subhashish Banerjee
Type: theory · Category: quantum information and computing · PDF: https://arxiv.org/pdf/2606.09379v1
Analysis basis: full PDF text, analyzed in chunks Topic relevance: entanglement & information structure 1/5 · quantum measurements 1/5

↔ Scroll figures horizontally

Summary. This paper develops novel entanglement-assisted continuous-variable concatenated codes by combining EA stabilizer codes with GKP bosonic codes. It theoretically demonstrates that these concatenated structures effectively suppress quadrature errors, achieving a variance reduction proportional to $1/n$. This advances the field of fault-tolerant quantum computation for bosonic systems.

Why it may be interesting. This work is highly relevant as it pushes the boundaries of quantum information theory by merging two advanced QEC paradigms (EA and CV bosonic codes). The focus on error suppression factors provides concrete theoretical benchmarks for future experimental implementations in quantum optics or superconducting circuits.

Detailed structure

Main problem. The research aims to enhance quantum error correction rates by combining entanglement-assisted (EA) stabilizer codes with bosonic continuous-variable (CV) codes, specifically the Gottesman-Kitaev-Preskill (GKP) code.

Main result. They propose and analyze two concatenated structures ($ ext{EA} riangleright ext{GKP}$ and $ ext{GKP} riangleright ext{EA}$) that demonstrate significant suppression of quadrature errors, achieving a variance reduction factor of $1/n$ in the general case.

Method. The analysis involves deriving logical state representations for concatenated codes, calculating syndrome measurements from Gaussian unitary encoders, and employing Maximum Likelihood Estimation (MLE) to estimate and correct residual quadrature errors.

Model / system. The system models quantum information encoded in bosonic modes using GKP states. The theoretical framework utilizes stabilizer formalism generalized to include entanglement assistance via ebits for error correction.

Key observables. Logical failure probability ($p_L$), residual quadrature error variances ($ ext{Var}(\zeta^_q) = ext{Var}(\zeta^p)$), and the success probability of bit identification ($p$).}

Important parameters / regimes. The number of qubits/modes ($n$), the noise variance ($\sigma^2$), and the entanglement resource (ebits).

Assumptions / limitations. The analysis assumes independent and identically distributed (iid) Gaussian random shift errors, and for the general result, it relies on syndrome measurements falling within a specific range $[-\sqrt{\pi}/2, \sqrt{\pi}/2]$.

Figures summary. Figure 1 illustrates encoding circuits for GKP Bell pairs using both computational and canonical bases. Figure 5 compares the logical failure probability versus standard deviation for different concatenation orders.

Paper structure. The paper first establishes the theoretical basis by combining EA concepts with bosonic codes (GKP). It then details two specific concatenation orders ($ ext{EA} riangleright ext{GKP}$ and $ ext{GKP} riangleright ext{EA}$) using examples like $n=3$. The core analysis involves deriving error suppression factors, culminating in a general theorem showing $1/n$ variance reduction for the $ ext{CGKP} riangleright C[[n,1,n;n-1]]$ code.

Abstract

Entanglement-assisted (EA) stabilizer codes enhance the rate of error correction in relation to codes with no pre-shared entanglement. Meanwhile, bosonic error-correcting codes, such as the Gottesman-Kitaev-Preskill (GKP) code, can be concatenated with qubit stabilizer codes to significantly reduce the logical failure probability of those stabilizer codes. First, we combine the above two concepts to propose an EA version of the qubit-into-oscillators concatenated code that chains an EA-stabilizer (outer) code with a GKP (inner) code. As an example we present a three-qubit EA-repetition concatenated with a GKP code. Second, we propose an EA version of the non-Gaussian oscillator-into-oscillators concatenated code that chains a GKP (outer) code with an EA-stabilizer (inner) code. As an example we present a GKP code concatenated with a three-qubit EA repetition code that uses two maximally entangled modes (emodes) and suppresses the variances of both position and momentum quadrature errors of a data mode. Furthermore, we generalize the latter example to a family of GKP code concatenated with a $n$-qubit EA repetition code that uses ${n-1}$ emodes and suppresses the variances of both position and momentum quadrature errors of a data mode by a factor ${1/n}$.

Exploring the landscape of compact magic-state distillation factories

Summary. This work tackles the critical problem of generating high-fidelity magic states necessary for fault-tolerant quantum computing. By reformulating distillation protocols using classical error-correcting codes, the authors used SAT solvers to derive new theoretical bounds and construct highly compact distillation factories. This advances the understanding of resource overhead by minimizing both qubit count and required operations.

Why it may be interesting. While focused on quantum computation theory, the rigorous application of classical coding theory and computational search techniques (SAT solving) provides powerful generalizable methods for analyzing resource overhead in complex physical systems.

Detailed structure

Main problem. The primary challenge is developing efficient, fault-tolerant methods for producing high-fidelity magic states (like $|T angle$) required for universal quantum computation.

Main result. The authors derived new, compact distillation protocols achieving higher distances ($d=5$ for T-to-T) using fewer physical qubits than previously known. They also established no-go theorems relating protocol parameters.

Method. They employed a combination of classical error-correcting code theory and Boolean Satisfiability (SAT) solvers to systematically constrain the design space of distillation protocols.

Model / system. The focus is on implementing logical T gates using magic state distillation factories based on the repetition code. The analysis involves mapping quantum constraints onto algebraic structures solvable by SAT solvers.

Key observables. Distillation distance ($d$), number of physical qubits ($N$), and the required T-count ($n$).

Important parameters / regimes. The relationship between $N$, $d$, and $n$ is central; specific results include achieving $d=5$ on $N=11$ qubits.

Assumptions / limitations. Protocols are analyzed by framing the problem using classical error-correcting code structures, simplifying the initial quantum description via Pauli product rotations.

Figures summary. Figures illustrate the comparison of T-count vs. number of logical qubits for various distillation distances and show circuit diagrams for gate teleportation/correction.

Paper structure. The paper progresses from general necessity (magic state distillation) to specific mathematical tools (classical codes, SAT solvers), deriving bounds and constructing optimized protocols for both $|T angle$ and $\sqrt{T}$ states.

Abstract

Producing high-fidelity magic states using the smallest possible amount of physical qubits and operations stands as a very important challenge to achieve fault-tolerant quantum computation at scale. Besides emerging proposals for alternative methods such as cultivation, magic state distillation remains essential for achieving very low error rates. Known distillation protocols are usually built through quantum codes derived from triorthogonal matrices. Here, exploiting the specific noise structure present in magic state distillation protocols, we show that classical error-correcting codes offer a simpler framework for deriving these protocols. This formulation is particularly well suited to systematic numerical and analytical studies of distillation protocols involving a fixed number of qubits. Specifically, we use a SAT solver to derive a series of no-go theorems that relate key figures of merit, including the number of qubits, the protocol depth, the factory distance, and the prefactor in the output error rate. For instance, we prove that any $T$-to-$T$ state distillation protocol using fewer than eight qubits can detect at most three errors, while any $T$-to-$\mathrm{CC}Z$ state distillation protocol using fewer than eight qubits can detect at most two errors. Our results also include new distillation protocols with the smallest number of qubits for a given distance in the literature, namely distance 4 and 5 $T$-to-$T$ state protocols supported on 10 and 11 qubits, as well as distance 3 and 4 $T$-to-$\mathrm{CC}Z$ state distillation protocols supported on 9 and 10 qubits.

Neural network decoder confidence as a learned proxy for the logical gap

Authors: David Dentelski
Type: theory · Category: quantum information and computing · PDF: https://arxiv.org/pdf/2606.08758v1
Analysis basis: full PDF text, analyzed in chunks Topic relevance: methods for driven-dissipative 1/5 · quantum measurements 1/5

↔ Scroll figures horizontally

Summary. This paper investigates whether the confidence score generated by a Graph Neural Network (GNN) can replace the traditional 'logical gap' used for decoding measurements in quantum error correction. By comparing performance on the rotated surface code, the authors demonstrate that the GNN-derived soft information leads to lower logical error rates and better calibration than the standard Minimum Weight Perfect Matching approach. This suggests GNNs are a powerful tool for extracting reliable confidence metrics from syndrome data.

Why it may be interesting. It provides a concrete example of applying machine learning techniques (GNNs) to extract physically meaningful soft information in quantum error correction, offering a scalable alternative when computationally intensive methods like MWPM are impractical.

Detailed structure

Main problem. To determine if a Graph Neural Network (GNN) decoder's raw output confidence score can serve as a reliable, learned proxy for the established 'logical gap' used in quantum error correction decoding.

Main result. Post-selection based on the GNN logit yields a lower logical error rate than using the conventional Minimum Weight Perfect Matching (MWPM) complementary gap across tested noise regimes. The GNN also shows better calibration towards ideal theoretical estimates.

Method. The study compares two soft information measures—the GNN's output logit and the MWPM gap—by testing their efficacy in post-selecting measurements from a quantum error-correcting code.

Model / system. The analysis is performed on the rotated surface code using syndrome measurements generated under uniform circuit-level noise. The comparison involves decoding performance across varying acceptance rates ($\kappa$) and physical error probabilities ($p$).

Key observables. Logical Error Rate (LER), signed confidence distribution, and calibration slope ($\alpha$) relative to the ideal posterior log-likelihood ratio.

Important parameters / regimes. Physical error rate ($p=0.005$), code distance ($d$), number of stabilizer rounds ($r$), and acceptance rate ($\kappa$).

Assumptions / limitations. The primary assumption is that ranking measurements by confidence score (rather than absolute scale) determines the post-selection advantage, and that the GNN can learn sufficient soft information without explicit knowledge of matching probabilities.

Figures summary. Figures compare LER vs. acceptance rate for both decoders; other figures show probability distributions and empirical calibrations of the signed confidence scores against theoretical expectations.

Paper structure. The paper introduces the problem by comparing GNN decoding to MWPM gap estimation, presents quantitative results showing superior performance (lower LER) for the GNN, analyzes the shape and calibration of the confidence scores, and concludes with implications for general decoder design.

Abstract

To utilize quantum error-correcting codes, a decoder must infer the logical sector from the measured syndrome. Beyond producing a hard logical decision, some decoders provide soft information that estimates the reliability of that decision. For minimum-weight perfect matching (MWPM), a common confidence measure is the complementary, or logical, gap. Here we test whether the logit of a graph neural network (GNN) decoder can act as a learned proxy for the logical gap. Using a pretrained GNN for the rotated surface code under uniform circuit-level noise [Physical Review Research, 7(2):023181, 2025], we compare its soft output with the MWPM complementary gap on the same sampled syndromes. We find that post-selection based on the GNN logit yields a lower logical error rate than one based on the MWPM gap. Shot-by-shot, the signed GNN confidence distribution resembles the signed MWPM gap at low and intermediate values, but assigns higher confidence to many correctly decoded shots. While both scores approximate the posterior log-likelihood ratio, the GNN confidence magnitude is closer to its ideal value. These results show that a neural-network decoder trained only on syndromes and logical labels learns both gap-like discrimination and a quantitative confidence scale, enabling confidence-based post-selection when MWPM gap estimates are unavailable, costly, or poorly matched to the noise model.

On the viability of Transatlantic Quantum Entanglement Distribution using Combined Satellite and Stratospheric Relay Nodes

Authors: Kimia Mohammadi, Paul J. Godin, Thomas Jennewein

Summary. This paper analyzes methods for achieving transatlantic quantum entanglement distribution using satellites and stratospheric relays. By modeling various link architectures, the authors determined that a hybrid LEO satellite and HAP system provides the highest secure key rate ($\approx 5 imes 10^6$ bits/year). This demonstrates the significant advantage of incorporating low-cost, maneuverable atmospheric relays over single high-altitude satellites.

Why it may be interesting. This work directly addresses the engineering challenges of scaling quantum communication networks over continental distances by proposing a practical hybrid space/air relay solution that significantly outperforms purely satellite-based approaches.

Detailed structure

Main problem. The study investigates feasible architectures for distributing quantum entanglement across transatlantic distances (6,500 km) to establish a global quantum network.

Main result. A hybrid architecture combining an entangled photon source on a Low Earth Orbit (LEO) satellite with two passive High-Altitude Platform (HAP) relays yields the highest entanglement distribution rate ($\approx 5 imes 10^6$ secure key bits/year).

Method. The authors modeled various free-space link configurations, comparing single MEO satellites against hybrid LEO+HAP systems by analyzing link budgets and achievable key rates.

Model / system. The system involves ground stations communicating with space assets (LEO/MEO satellites) and stratospheric HAPs acting as passive optical relays. The analysis incorporates orbital mechanics, free-space link loss calculations, and quantum key distribution metrics.

Key observables. Entanglement distribution rate (secure key bits per year), Quantum Bit Error Rate (QBER), and violation of the CHSH Bell inequality parameter ($S$).

Important parameters / regimes. Transatlantic distance ($\sim 6,500$ km); LEO altitude ($\sim 1,200$ km); HAP altitude ($\sim 20$ km); Ground receiver aperture sizes (e.g., $30 ext{ cm}$ vs $1 ext{ m}$).

Assumptions / limitations. The study assumes that HAPs act purely as passive relays and does not rely on quantum memories or repeaters for entanglement distribution.

Figures summary. Figures compare link loss and elevation angle variations across different satellite constellations (MEO vs. LEO+HAP); Figure 7 illustrates the relationship between Bell-violation parameters and QBER requirements.

Paper structure. The paper systematically compares multiple link architectures (single MEO, multi-LEO, LEO+HAP) by calculating key performance indicators like achievable secure key rates and verifying entanglement preservation via Bell tests.

Abstract

To explore the pathways toward establishing a global quantum network, we investigate several link architectures for transatlantic quantum entanglement distribution over a 6,500 km ground distance. We define free-space link configurations involving satellites and stratospheric high altitude platforms (HAPs), using today's technology and without relying on quantum memories and repeaters. Considering link budgets, space radiation, orbital characteristics, and system complexity we find that a hybrid architecture consisting of an entangled photon source located on a low Earth orbit (LEO) satellite supported by two passive optical relays located on HAPs provides the overall highest entanglement distribution rate. In addition, the satellite HAP architecture offers practical advantages in payload design and launch requirements, and the ability to lower the weather-related link interruptions assuming some maneuverability of HAPs. Overall, this hybrid configuration yields on the order of 5X10^6 secure key bits per year using 30 cm aperture ground receivers, nearly two orders of magnitude higher than achievable with a single MEO satellite and 1 m aperture ground receivers. Our results highlight the major benefits of hybrid satellite HAP architectures by reducing system complexity while enabling scalable and more accessible long-range quantum communication networks.

Probabilistically Checking Quantum Proofs, with Interaction

Authors: Baocheng Sun, Thomas Vidick
Type: theory · Category: quantum information and computing · PDF: https://arxiv.org/pdf/2606.09588v1
Analysis basis: full PDF text, analyzed in chunks Topic relevance: entanglement & information structure 1/5 · quantum measurements 1/5

↔ Scroll figures horizontally

Summary. This work develops a quantum interactive proof system (qIOP) to characterize problems in the complexity class QMA. By combining quantum locally testable codes with classical PCP techniques, the authors show that QMA can be verified using only polylogarithmic measurements from the verifier. This result is significant as it provides an information-theoretic characterization of QMA without assuming a quantum PCP theorem.

Why it may be interesting. It provides deep connections between complexity theory (QMA) and quantum coding theory, offering tools for verifying complex quantum computations efficiently.

Detailed structure

Main problem. The paper aims to establish a quantum analogue of interactive oracle proofs (qIOP) to characterize Quantum Merlin-Arthur (QMA) problems.

Main result. They prove that every language in QMA possesses an efficient qIOP where the verifier only needs to read a polylogarithmic number of qubits, providing a local and robust characterization of QMA.

Method. The protocol combines quantum Locally Testable Codes (qLTCs) with classical techniques like Probabilistically Checkable Proofs of Proximity (PCPP). The verification process involves multiple stages: measurement extraction and constraint checking.

Model / system. The model is the qIOP, involving a quantum verifier interacting with an unbounded prover. It utilizes quantum error-correcting codes (qLTCs) to encode states, which are tested via local measurements on small subsets of qubits.

Key observables. Acceptance probability of the energy test ($ ext{Pr}[ ext{energy test accepts}]$), fidelity between state evolution operators, and measurement outcomes in Z or X bases.

Important parameters / regimes. The verifier reads a polylogarithmic number of qubits ($q$), while total communication is polynomial ($\ell$). The success relies on the code distance ($d_q$) and locality ($q$).

Assumptions / limitations. The result holds in the absence of a quantum PCP theorem, relying on the existence of suitable qLTC codes with specific parameters.

Paper structure. The paper builds from defining the qIOP model to implementing measurement extraction using qLTCs (Single/Global Measurement Extraction), followed by a final energy test phase that combines these results to prove the QMA characterization.

Abstract

The model of interactive oracle proofs (IOP) generalizes the notion of probabilistically checkable proof (PCP), in which a static proof is verified probabilistically by querying a small number of bits, to the interactive setting: a polynomial-time verifier interacts with an unbounded prover, but is restricted to only reading a small number of bits, in total, from the messages sent by the prover. IOPs provide a relaxed setting in which to study local probabilistic verification. They have proved instrumental in devising efficient methods for verification through subsequent compilation into non-interactive or succinct protocols. We study a quantum analogue of interactive oracle proofs (qIOP) in which the verifier and communication are both allowed to be quantum; yet the verifier is restricted to perform measurements only on a small number of qubits received from the prover. Our main result is a qIOP for any language in QMA, in which the total communication is polynomial but the verifier only reads a polylogarithmic number of qubits in total. The protocol has completeness parameter exponentially close to $1$ and soundness bounded away from $1$ by a constant. In the absence of a quantum PCP theorem, this provides the first information-theoretically sound local and robust characterization of QMA, albeit interactive. Our protocol combines the use of a quantum locally testable code (LTC) with classical techniques, notably probabilistically checkable proofs of proximity (PCPP). We avoid the necessity for complex multi-qubit tests employed in other settings by leveraging the local indistinguishability property of the quantum LTC.

Superconductivity in the high-pressure tetragonal phase of UTe2

Authors: Yuhang Deng, Gabriel Mee, Tyler Wannamaker, Keke Feng, Mingyu Xu, Weiwei Xie, M. Brian Maple
Type: experiment · Category: strongly correlated electrons · PDF: https://arxiv.org/pdf/2606.07903v1
Analysis basis: full PDF text, analyzed in chunks Topic relevance: Kondo & dissipative impurity 1/5 · analog quantum simulation 1/5

↔ Scroll figures horizontally

Summary. This work experimentally maps the superconducting phase diagram of UTe2 under high pressure, revealing a striking dichotomy between its orthorhombic and tetragonal phases. The findings suggest that structural changes tune the pairing mechanism from an unconventional state to a more conventional one in the tetragonal phase. This highlights how external tuning can stabilize different types of correlated electronic ground states.

Why it may be interesting. While focused on condensed matter, the comparison of BCS-like vs. unconventional pairing symmetries under external tuning (pressure) provides a model for understanding how quantum critical points can stabilize different types of correlated ground states.

Detailed structure

Main problem. The study aims to map out the superconducting transition temperature ($T_c$) versus pressure ($P$) phase diagram of UTe2, specifically investigating the structural and electronic dichotomy between its orthorhombic and high-pressure tetragonal phases.

Main result. Superconductivity in the tetragonal phase is robust and appears more conventional (orbitally limited) compared to the fragile, unconventional superconductivity seen in the orthorhombic phase. The normal state exhibits a persistent 'knee' feature ($T^{**}$) linked to potential magnetic ordering.

Method. The research relies on electrical transport measurements ($ ext{R}(T)$) and AC magnetic susceptibility ($\chi(T,P)$) performed under high pressure using Diamond Anvil Cells (DACs).

Model / system. The system is Uranium ditelluride ($ ext{UTe}_2$), a heavy-fermion superconductor known for complex interplay between structural transitions, magnetism, and superconductivity. The study compares the superconducting behavior across two distinct crystal structures.

Key observables. Superconducting transition temperature ($T_c$), upper critical field ($ ext{H}_{c2}$), normal-state resistivity $ ho(T)$ exhibiting a 'knee' feature ($T^{**}$).

Important parameters / regimes. Pressure range up to 16 GPa; characteristic fields like the Pauli paramagnetic limit and orbital limits.

Assumptions / limitations. The analysis assumes that structural changes (orthorhombic $\leftrightarrow$ tetragonal) fundamentally alter the pairing mechanism, leading to different superconducting symmetries.

Figures summary. Figures illustrate representative resistance vs. temperature curves showing SC transitions in the tetragonal phase at various pressures, and potentially show the $T-P$ phase diagram.

Paper structure. The paper systematically measures transport properties across structural transitions, contrasting the behavior of superconductivity (SC) and normal state features ($T^{**}$) between the two phases to deduce underlying electronic mechanisms.

Abstract

Electrical transport and magnetic measurements have been made on UTe2 under pressure P up to approximately 16 GPa to determine the superconducting transition temperature Tc vs P phase diagram in the high-pressure tetragonal phase. Superconductivity emerges near 5 GPa, coincident with the orthorhombic to tetragonal phase transition; in the tetragonal phase, Tc reaches a maximum value of approximately 4 K at 6 GPa and then decreases with P and appears to vanish near 18 GPa. Tetragonal UTe2 has a relatively small upper critical field Hc2(0) $\approx$ 1.2 T at 5.3 GPa, smaller than the Pauli paramagnetic limit, and is orbitally limited with a coherence length $ξ$tetra $\approx$ 16.5 nm. This small value of Hc2(0) favors more conventional superconductivity; in contrast, the large values of Hc2(T) for orthorhombic UTe2 exceed the Pauli paramagnetic limit in all three crystallographic directions and have been attributed to unconventional superconductivity, widely believed to involve spin-triplet pairing. The temperature-pressure phase diagram of UTe2 shows a striking dichotomy: a narrow, fragile, unconventional superconducting region in the orthorhombic phase vs a broad, robust, and more conventional superconducting dome in the tetragonal phase. This dichotomy is consistent with the proposal that U-dimers, present (absent) in the orthorhombic (tetragonal) phase, may play a role in spin-triplet superconductivity of orthorhombic UTe2. In the tetragonal phase, the normal-state electrical resistivity $ρ$(T) exhibits metallic behavior with a knee between 200 and 240 K that depends weakly on P and most likely marks the onset of a transition to a magnetically ordered phase that coexists with superconductivity.

Adaptive directional gradients for parameterised quantum circuits

Authors: Brian Coyle, Snehal Raj, Virag Umathe, El Amine Cherrat, Elham Kashefi
Type: both · Category: quantum information and computing · PDF: https://arxiv.org/pdf/2606.09734v1
Analysis basis: full PDF text, analyzed in chunks Topic relevance: quantum measurements 1/5

↔ Scroll figures horizontally

Summary. The paper introduces an advanced forward gradient estimator framework to overcome the prohibitive measurement cost associated with calculating gradients for large parameterised quantum circuits. By adaptively optimizing both the number of random directions and the shots allocated per direction, the proposed QUIVER optimizer significantly outperforms established methods across various quantum machine learning benchmarks.

Why it may be interesting. This work provides a concrete, resource-aware algorithmic improvement for variational quantum algorithms, directly addressing the primary computational bottleneck in scaling up near-term quantum hardware applications.

Detailed structure

Main problem. Training parameterised quantum circuits (PQCs) is bottlenecked by the measurement cost of gradient estimation, which scales poorly with the number of trainable parameters.

Main result. A forward gradient estimator framework, combined with an adaptive optimizer (QUIVER), achieves orders of magnitude more efficient training than standard methods like the Parameter-Shift Rule for large quantum neural networks.

Method. The authors propose a forward gradient estimator based on the forward mode of automatic differentiation, averaging random directional derivatives to reconstruct the full gradient efficiently. They combine this with an adaptive optimizer that jointly optimizes the number of directions and measurement shots.

Model / system. The work focuses on training PQCs used in Variational Quantum Algorithms (VQA) like VQE and QNNs. Benchmarks include classifying ECG5000 data, MNIST images, solving QAOA MaxCut problems, and estimating ground states for TFIM Hamiltonians.

Key observables. Energy error ($|E - E_0|$), approximation ratio ($r$), Mean Squared Error (MSE) of the gradient estimator, and total measurement shot count.

Important parameters / regimes. Number of parameters ($N$), number of random directions sampled ($V$), shots per direction ($M$), step size ($\epsilon$), and learning rate ($\eta$).

Assumptions / limitations. The analysis relies on assumptions regarding the convergence properties (e.g., PL condition locally) and uses central finite-difference approximations for directional derivatives, which introduce bias.

Figures summary. Figures compare the measurement shot costs of various gradient estimators (PS vs. Forward) showing significant savings with forward gradients; other figures show performance comparisons across multiple benchmark tasks (QAOA, VQE, MNIST) demonstrating superior convergence rates for QUIVER.

Paper structure. The paper systematically reviews existing gradient estimation techniques (PS, SPSA, RCD), introduces the unified forward gradient estimator parameterized by $V$, develops the adaptive optimization strategy (QUIVER) via joint $(V, M)$ minimization, and validates this approach across multiple complex quantum machine learning benchmarks.

Abstract

Training parameterised quantum circuits (PQCs) on quantum hardware is bottlenecked by the measurement cost of gradient estimation, which under the parameter-shift rule scales linearly in the number of trainable parameters and dominates the total shot budget of training at scale. In this work, we propose a framework of forward gradient estimators for PQCs, based on the forward mode of automatic differentiation, that yields an unbiased estimator of the gradient by averaging a freely tunable number of random directional derivatives and recovers SPSA, random coordinate descent, and the parameter-shift rule as limiting cases, with no ancilla qubits or controlled-gate overhead. We prove that stochastic quantum forward gradient descent converges under standard assumptions, with an explicit second-moment expansion that interpolates between the single-direction extreme of SPSA and the full-gradient extreme of parameter-shift. Within this framework we derive QUIVER (Quantum Iterative V-adaptive Estimator Rule), an adaptive optimiser for parameterised circuits whose update rule follows from a closed-form minimum measurement-cost allocation. We show numerically that forward gradients train Hamming-weight-preserving orthogonal quantum neural networks with up to 60 qubits and 1770 parameters on the ECG5000 and MNIST datasets orders of magnitude more efficiently than the parameter-shift rule. We also demonstrate that our proposed QUIVER optimiser can outperform iCANS and gCANS measurement-frugal optimisers on optimisation problems using the quantum approximate optimisation algorithm and quantum simulation with the variational quantum eigensolver.

Summary. This paper explores alternative geometric foundations for quantum gravity by focusing on teleparallel gravity, which replaces curvature with torsion as the fundamental field strength. It argues that using coframe and spin-connection variables provides a natural gauge structure suitable for quantization. The work aims to identify necessary conceptual ingredients rather than providing a complete solution to quantizing spacetime itself.

Why it may be interesting. While focused on gravity, the development of a constrained quantum field theory based on gauge potentials (coframe/spin-connection) shares conceptual parallels with advanced topics in quantum information theory involving gauge symmetries and constraints.

Detailed structure

Main problem. The fundamental incompatibility between Quantum Field Theory (QFT) and General Relativity (GR) at the Planck scale, necessitating a new framework for quantizing gravity.

Main result. The paper proposes that reformulating gravity using coframe and spin-connection variables, where dynamics are governed by torsion rather than curvature, provides a geometrically refined starting point for quantum gravity research.

Model / system. Important parameters / regimes. Planck scale; coupling constants $\kappa$; mass $m$ (for Proca field).

Figures summary. Not specified in the notes; the text references comparative tables (TABLE I) summarizing different QG approaches based on their fundamental variables.

Abstract

We revisit quantum field theory in curved spacetime (QFTCS) as a semi-classical framework for quantum matter on classical geometries, emphasizing its limitations, including vacuum ambiguity and background dependence. We briefly review major approaches to quantum gravity (QG), including Loop Quantum Gravity (LQG), string theory, and asymptotic safety, highlighting their conceptual challenges. Motivated by these issues, we outline a teleparallel framework based on coframe and spin-connection variables, where gravity is encoded in torsion rather than curvature. This framework naturally incorporates local Lorentz symmetry and fermionic couplings while displaying a gauge-like structure. We argue that the coframe/spin-connection pair provides an alternative and geometrically refined description of gravitational variables, which may serve as a useful starting point for future investigations of QG. The purpose of this work is not to provide a complete quantization of teleparallel gravity but to identify the geometric and conceptual ingredients that such a formulation would require.

Summary. This work develops a theoretical framework to explain excess specific heat observed below the Debye prediction in perfect crystals. It attributes this anomaly to energy fluctuations caused by anharmonic interactions involving next-nearest-neighbor atoms modulating harmonic lattice vibrations. The theory successfully models these deviations using molecular dynamics simulations and provides insights applicable to amorphous materials.

Why it may be interesting. The connection drawn between lattice dynamics, anharmonicity, and anomalous low-temperature thermodynamics is highly relevant for understanding energy transport in complex quantum systems.

Detailed structure

Main problem. The paper addresses the discrepancy between measured specific heat at low temperatures and the predictions of Debye's $T^3$-law in high-purity crystals.

Main result. Excess energy fluctuations, arising from anharmonic interactions involving next-nearest-neighbor atoms, modify the standard relation between energy fluctuations and specific heat, providing a quantitative explanation for the observed deviations.

Based on these results, we formulate a prescription for constructing BBS codes that works for regions with rectangular, diagonal, and arbitrarily tilted boundaries. A key advantage of this approach is that no corner corrections are needed, provided the polynomials satisfy orientation-specific edge conditions.

Summary. This paper introduces Quantum-Native Reinforcement Learning (QnRL), a framework that overcomes the limitations of classical and prior quantum RL by representing probability distributions as complex quantum states. It utilizes a novel QuAK algorithm to analyze these distributions directly in Hilbert space, leading to superior performance across diverse simulated environments.

Why it may be interesting. It provides a concrete methodology for embedding complex statistical learning problems (like DRL) directly into quantum circuit operations, offering insights into how quantum information theory can solve computational bottlenecks in AI.

Detailed structure

Main problem. Existing Quantum Reinforcement Learning (QRL) methods limit expressive power by only estimating expected outcomes, failing to model the full probability distributions governing stochastic environments.

Main result. The proposed QnRL framework achieves significantly higher evaluation scores and requires fewer parameters than baselines by natively modeling conditional return distributions in Hilbert space.

Method. It introduces Quantum-Native Reinforcement Learning (QnRL), which uses a novel Quantum Amplitude Kickback (QuAK) algorithm to analyze moments of superimposed quantum state distributions directly within the complex Hilbert space.

Model / system. The system extends classical DRL by representing return and policy distributions as superimposed, entangled quantum states in a combined action-return Hilbert space ($H_A \otimes H_Z$). The core mechanism involves Variational Quantum Circuits (VQC) for distribution generation.

Key observables. Evaluation scores, expected total reward, conditional action policy distribution $P(A|x)$, and the ability to capture complex correlations via phase/amplitude components.

Important parameters / regimes. Action qubits ($q_A$), return qubits ($q_Z$), moment order ($m$), expectation exponent ($n$), circuit depth ($L$).

Assumptions / limitations. The primary assumption is that quantum computation can provide an advantage by keeping the entire distributional learning process within the complex Hilbert space, avoiding classical measurement sampling loss.

Figures summary. Figures illustrate the four core stages of QnRL: Return distribution generation (superposition), moment analysis via QuAK, action selection based on moments, and final model optimization.

Paper structure. The paper details the system model expansion from classical to quantum settings, introduces the QnRL framework and the QuAK algorithm in dedicated sections, presents comprehensive results across multiple challenging environments, and concludes with detailed hyperparameter comparisons.

Abstract

Quantum reinforcement learning (QRL) is a promising approach to learn effective decision strategies across several applications with stochastic environments. Instead of directly modeling the random variables that govern these environments, existing QRL architectures indirectly approximate environment behavior by estimating expected outcomes, which limits their expressive power and adaptive potential. Overcoming such challenges requires a novel QRL approach that exploits the distributional nature of quantum computers to directly model environment random variables as quantum state distributions. Hence, in this paper, a novel framework dubbed quantum-native reinforcement learning (QnRL) is proposed. QnRL is a distributional RL framework that learns conditional distributions naturally in Hilbert space via superimposed and entangled quantum states. Thus, QnRL can directly model the behavior of stochastic learning environments via the natural properties of quantum systems. QnRL accomplishes this via a novel, proposed quantum amplitude kickback (QuAK) algorithm that enables comparing the $n$-th power of the $m$-th moment of multiple superimposed distributions. It is theoretically proven that a conditional action policy distribution is distilled from the moments of a quantum generative model entirely within Hilbert space via QuAK, and optimized via QnRL. This complex distribution composition is also shown to provide extra dimensions for expressing environment correlations that are unknown to purely classical and classically-sampled quantum distributional models. Experimental results across diverse environments show that QnRL achieves up to $82.9\%$ higher evaluation scores, with up to $94.3\%$ fewer parameters on average, more accurately estimates the expected return for unseen observations, and better adapts to varying stochastic conditions compared to the baseline.
